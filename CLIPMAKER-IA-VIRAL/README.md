# 🎥 ClipMaker IA Viral

> Projeto desenvolvido durante o **NLW** da Rocketseat, explorando o uso de **Inteligência Artificial** no front-end para criação de clips virais.

---

## ✨ Funcionalidades

- 🤖 Integração com IA para geração de conteúdo
- 🎬 Interface para criação e edição de clips
- 💡 Operações assistidas por IA em front-end

---

## 🛠️ Tecnologias utilizadas

| Tecnologia | Uso |
|---|---|
| HTML5 | Estrutura da aplicação |
| CSS3 | Estilização |
| JavaScript | Lógica e integração com IA |
| IA (NLW Operador) | Geração de conteúdo |

---

## 🚀 Como rodar localmente

```bash
git clone https://github.com/Luiis-dev/CLIPMAKER-IA-VIRAL.git
cd CLIPMAKER-IA-VIRAL
# Abra o index.html no navegador ou use Live Server no VS Code
```
